Programming Inteview Questions API

Welcome to Programming Interview Questions API.

Here you will find a free api that allows you to retreive programming focused interview questions and solutions. I have set up four different api endpoints that currently only host a total of 10 questions each. The motivation for creating this API was the inability to find my own when looking for a free one to use. More questions will be added in the future.

The types of endpoints include:

  • /programming-question
  • /programming-questions
  • /star-question
  • /star-questions

/programming-question

The programming-question endpoint can be reached via Programming Question Endpoint

The route in which it can be achieved is

/api/programming-question

This route uses the GET method http method and returns a programming interview question, and a link to a solution in java in JSON format.

/programming-questions

The programming-questions endpoint can be reached via Programming Questions Endpoint

The route in which it can be achieved is

/api/programming-questions

This route uses the GET method http method and returns all of the programming interview questions, and a link to all of the solutions in java in JSON format.

/star-question

The star-question endpoint can be reached via Star Question Endpoint

The route in which it can be achieved is

/api/star-question

This route uses the GET method http method and returns a star behavioral interview question in JSON format.The motivation for creating this endpoint was that programming interviews are not only technical but behavioral as well. This will help in preparing for interviews.

/star-questions

The star-questions endpoint can be reached via Star Questions Endpoint

The route in which it can be achieved is

/api/star-questions

This route uses the GET method http method and returns all of the star behavioral interview questions in JSON format.The motivation for creating this endpoint was that programming interviews are not only technical but behavioral as well. This will help in preparing for interviews.
